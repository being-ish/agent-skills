# 日本語 Markdown スタイルガイド

日本語で Markdown 文書を書く、直すときの指針。

機械的に白黒を付けられる規則は同プラグインの PostToolUse Hook が検査する。
このスキルは、Hook では判定できない、書き手の判断が必要な指針を扱う。

## 文章構成の指針

木下是雄「理科系の作文技術」に則る。

- 結論を先に短く書き、逆茂木型の文章を避ける
- 括弧での補足を使わず、必要な情報は地の文に書く
- 同じ情報を複数個所に書かない
    - 前後のつながりが自然なところに書き、他からは参照する

## 書き方の指針

- 造語を使わない
    - 正式な用語か一般的な言葉のみ使う
- カタカナ外来語の表記は総務省準拠とする
    - 末尾長音を付ける
- 出典は末尾に列挙せず、本文の地文にインラインリンクを張る
    - インラインリンクを張るときは出典名をアンカーテキストにしない
- 対応関係は表にする
- 列挙できる内容は箇条書きを使う
    - 箇条書きのネストのインデントは半角スペース 4 とする
- `**太字**` を基本使わない
    - 強調しないと想定読者に意図が伝わらない箇所だけ使う
    - コード識別子は `バッククォート` で足りる
